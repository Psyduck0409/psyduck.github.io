RoarCTF Crypto

Crypto_System

考虑到 $s = g^{m+br}$， 要有$s1 == s2$，就是$m_1 + b_1 r_1 = m_2 + b_2 r_2 \mod phi(p)$，看了翅膀师傅wp知道这里更准确应该用$m_1 + b_1 r_1 = m_2 + b_2 r_2 \mod q$求出r2(其中q是g模p的阶,可用sage求出)。

easyRSA

分析一下tip可求,$tip \mod beta = x + y \mod beta$, $phi = (tip - x - y)2beta$。于是爆破$x + y$即可。

Reverse

赛后翅膀师傅提示参考ASIS2020 RSASR，等于是前后一起爆破p和Reverse(p)。
